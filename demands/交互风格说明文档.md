# 语习集 - 交互风格说明文档

## 1. 设计概述

### 1.1 产品定位
语习集是一款英语口语学习应用，采用场景化学习方式，帮助用户在真实对话场景中掌握实用口语表达。

### 1.2 设计理念
- **简洁现代**：采用扁平化设计，界面清爽，降低学习干扰
- **场景沉浸**：通过场景化卡片和对话气泡，营造真实学习氛围
- **友好引导**：使用柔和的色彩和圆角设计，降低学习焦虑
- **即时反馈**：通过进度环、评分等可视化元素，提供学习成就感

## 6. 动效规范

### 6.1 过渡效果
```css
/* 卡片悬停 */
transition: all 0.3s ease;
transform: translateY(-2px);

/* 按钮点击 */
transform: scale(0.98);

/* 弹窗 */
transition: transform 0.3s ease;
```

### 6.2 动画效果

#### 录音动画
```css
animation: pulse 1.5s infinite;
@keyframes pulse {
  0%, 100% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.1); opacity: 0.7; }
}
```

#### 加载动画
```css
animation: spin 1s linear infinite;
@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}
```

#### 录音指示器
- 三个圆点依次脉冲动画
- 动画延迟：0s, 0.2s, 0.4s

## 8. 交互模式

### 8.1 点击反馈
- 卡片点击：`scale(0.98)` 后恢复
- 按钮点击：视觉状态变化
- 列表项：整行可点击

### 8.2 状态切换
- 标签切换：背景色 + 文字色变化
- 选项选择：边框高亮 + 填充标记
- 导航激活：主色高亮

### 8.3 录音交互
- 点击录音按钮 → 显示脉冲动画
- 录音中 → 红色指示器 + 波形动画
- 录音完成 → 结果显示 + AI评价

### 8.4 筛选交互
- 点击筛选按钮 → 底部弹窗滑入
- 选择条件 → 实时预览
- 应用筛选 → 列表刷新

## 12. 使用建议

1. **保持一致性**：所有卡片使用统一的圆角和阴影
2. **色彩语义化**：使用功能色表达状态，场景色表达分类
3. **动效克制**：过渡动画保持 0.2-0.3s，避免过度动画
4. **触摸友好**：确保所有可点击元素有足够大的触摸区域
5. **视觉层次**：通过间距、字号、色彩建立清晰的信息层级
