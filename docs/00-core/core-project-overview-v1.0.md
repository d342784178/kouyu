## 项目背景

### 问题定义

在中国，尽管英语学习者众多，但"哑巴英语"现象依然普遍。传统英语学习应用存在以下痛点：

1. **场景脱离实际**: 学习内容与实际生活场景脱节
2. **缺乏口语练习**: 侧重读写，忽视听说能力培养
3. **互动性不足**: 缺少真实的对话练习环境
4. **反馈不及时**: 无法获得即时的发音和表达纠正

### 解决方案

语习集是一款聚焦日常常用英文短语学习的AI驱动型移动端Web应用，通过以下方式解决上述问题：

- **真实场景**: 110+真实生活场景对话（餐厅、机场、酒店等）
- **AI对话**: 基于GLM-4-Flash的智能对话模拟
- **语音合成**: Azure Speech SDK提供高质量语音
- **即时反馈**: AI实时评估对话表现

## 项目目标

### 短期目标

- 提供覆盖日常生活、职场、留学、旅行、社交五大场景的学习内容
- 建立111个场景对话，每个场景包含完整对话流程和高频词汇
- 实现AI驱动的开放式对话练习功能

### 长期目标

- 打造最贴近真实场景的英语口语学习平台
- 通过AI技术实现个性化学习路径推荐
- 建立用户学习数据追踪和进步可视化系统

## 核心功能

### 1. 场景学习

| 功能 | 说明 |
|------|------|
| 场景列表 | 按分类浏览所有学习场景 |
| 场景详情 | 查看完整对话内容、词汇、翻译 |
| 音频播放 | 支持对话和词汇的语音播放 |
| 学习进度 | 记录场景学习状态 |

### 2. 短语库

| 功能 | 说明 |
|------|------|
| 短语列表 | 浏览高频实用短语 |
| 短语详情 | 查看翻译、发音提示、例句 |
| 分类筛选 | 按场景、难度筛选短语 |

### 3. AI对话练习

| 功能 | 说明 |
|------|------|
| 角色选择 | 选择用户角色和AI角色 |
| 难度设置 | 初级/中级/高级 |
| 多轮对话 | 支持5-8轮连续对话 |
| 语音交互 | 支持语音输入和AI语音回复 |
| 对话分析 | 结束后提供表现分析 |

### 4. 测试评估

| 功能 | 说明 |
|------|------|
| 场景测试 | 针对学习场景的练习题 |
| 开放式测试 | AI驱动的对话测试 |
| 填空测试 | 词汇和短语填空练习 |

                              │

## 目标用户

### 主要用户群体

1. **职场人士**: 需要提升商务英语沟通能力
2. **留学生**: 准备出国或已在国外学习生活
3. **旅行爱好者**: 需要旅行场景英语表达
4. **英语自学者**: 希望系统提升口语能力

### 用户画像

| 属性 | 描述 |
|------|------|
| 年龄 | 18-35岁 |
| 英语水平 | 初级到中级 |
| 学习目的 | 实用口语能力提升 |
